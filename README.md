# Survey courses backend - NestJS + TypeScript + TypeORM

## Цель проекта

Бэкенд, предоставляющий данные о курсах и опросы для записи на них.

- [X] Система регистрации и авторизации по почте
- [X] Управление профилем и детьми пользователя
- [X] Подача заявок на занятия с заполнением анкеты
- [X] Управление заявками администратором (одобрение, отклонение, блокировка)

- [X] Управление оплатой записей
- [X] Справочники (категории занятий)
- [X] Swagger документация
- [ ] Загрузка изображений для занятий

## Технологии

- **TypeScript** — статическая типизация для повышения надёжности кода
- **NestJS** — фреймворк для построения серверных приложений
- **TypeORM** — ORM для работы с базой данных
- **MariaDB** — реляционная база данных
- **Passport.js + JWT** — аутентификация и авторизация
- **Nodemailer + Handlebars** — отправка писем с шаблонами
- **Argon2** — хэширование паролей и токенов
- **Swagger** — документация API
- **Docker** — контейнеризация БД и почтового сервера

## Установка

1. Клонировать репозиторий
```bash
git clone 
cd survey-courses-backend
```

2. Установить зависимости
```bash
npm install
```

3. Создать `.env` файл по образцу `.env.example`

Для генерации JWT секретов выполните команду дважды (для `JWT_ACCESS_SECRET` и `JWT_REFRESH_SECRET`):

```bash
node -e "console.log(require('crypto').randomBytes(256).toString('base64'))"
```

4. Запустить БД и почтовый сервер через Docker
```bash
docker-compose up -d
```

### Режим разработки

5. Запустить приложение
```bash
npm run start:dev
```

### Продакшен

5. Скомпилировать приложение
```bash
npm run build
```

6. Запустить приложение
```bash
npm run start
```

API сервера доступен по адресу: `http://{HOST}:{PORT}/api/v1`  
Swagger документация доступна по адресу: `http://{HOST}:{PORT}/api/v1/docs`

## Структура БД

| Таблица | Описание |
|---|---|
| `users` | Пользователи (родители, модераторы, администраторы) |
| `children` | Дети пользователя |
| `code_verifications` | Коды подтверждения почты и сброса пароля |
| `refresh_sessions` | Сессии refresh токенов |
| `categories` | Категории занятий (справочник) |
| `lessons` | Занятия / курсы |
| `lesson_weekly_slots` | Типовое расписание занятия по дням недели |
| `lesson_schedule_overrides` | Исключения в расписании (отмена, перенос) |
| `lesson_pricing_tiers` | Тарифы занятия (разовое, абонемент) |
| `lesson_images` | Изображения занятия |
| `surveys` | Опросы к занятиям |
| `questions` | Вопросы опроса |
| `question_options` | Варианты ответов на вопросы |
| `applications` | Заявки на занятия (с анкетой) |
| `answers` | Ответы на вопросы анкеты |
| `enrollments` | Записи на занятия (абонементы) |
| `attendances` | Посещения занятий |
